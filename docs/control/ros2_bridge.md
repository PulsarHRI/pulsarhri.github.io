# ROS 2 Bridge

!!! warning "Under development"
    The PULSAR HRI ROS 2 Bridge is under development and is not published yet.

The ROS 2 Bridge is intended to expose PULSAR actuator control through ROS 2 interfaces while reusing the same actuator control modes documented in [Control Modes](control_modes_explained/control_modes_overview.md).

For currently supported workflows, use the [PULSAR App GUI](pulsar_app/pulsar_app.md), [Python API](python_api/install_python_api.md), or [C++ API](cpp_api/install_cpp_api.md).
