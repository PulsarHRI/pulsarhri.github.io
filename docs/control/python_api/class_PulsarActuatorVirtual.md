# PulsarActuatorVirtual

`PulsarActuatorVirtual` is the canonical API 2.0.0 class for running virtual PULSAR actuators backed by AUGUR Digital Twin assets. The first public DTwin beta asset release is available for Linux x86_64 and Windows x86_64; see the [download page](../../download/download_dtwin.md) for the current package and manifest.
