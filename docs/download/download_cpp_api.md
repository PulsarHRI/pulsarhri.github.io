# Download C++ API

## Available Packages

PULSAR HRI provides the C++ API in two Linux package formats:

| Package | Recommended for | Download |
|---|---|---|
| Debian package | Ubuntu/Debian users | [pcp-api-cpp-0.9.1-linux-x86_64.deb](pcp-api-cpp-0.9.1-linux-x86_64.deb) |
| Tarball SDK | Manual/custom installation | [pcp-api-cpp-0.9.1-linux-x86_64.tar.gz](pcp-api-cpp-0.9.1-linux-x86_64.tar.gz) |

Use the `.deb` package if you are using Ubuntu or Debian. Use the `.tar.gz` archive if you want a manual installation in a custom location.
